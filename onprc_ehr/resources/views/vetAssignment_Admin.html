Vet Assignment Hierarchy

Vets are assigned to NHPs according to the first rule listed below that matches.

  1. Open Case
  2. Room + Priority
  3. Area + Priority
  4. Project* + Room + Priority
  5. Project* + Room
  6. Project* + Area + Priority
  7. Project* + Area
  8. Project* + Priority
  9. Project*
  10. Protocol + Room + Priority
  11. Protocol + Area + Priority
  12. Protocol + Room
  13. Protocol + Area
  14. Protocol + Priority
  15. Protocol
  16. Room
  17. Area

* Research projects are automatically prioritized above non-research projects for the same rule.
